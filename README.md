# AURELIA — Fine Art Jewelry

A single-page, static demo marketing website for a fictional luxury jewelry brand — *"Timeless artifacts for the modern muse."* Built with plain HTML, CSS, and vanilla JavaScript (no framework or build step).

**Live site:** https://mbilalsiddiqi.github.io/jewelry_web1/

## Sections
Hero · The Collection (gallery) · Testimonials · Contact (Inquire) · Footer — all on `index.html`.

## Tech
- HTML5, CSS3 (custom properties, IntersectionObserver fade-ins), vanilla JavaScript
- Aesthetic: minimalist luxury — off-white `#F5F0EB`, gold accent `#C7A86B`
- Google Fonts: Cormorant Garamond + Inter
- Shared `styles.css` and `script.js`

## Run locally
No build step required — open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000   # visit http://localhost:8000
```
